บริสุทธิ์ ล่วงโสตของมนุษย์ ดูกรผู้มีอายุ ภิกษุใดรู้อย่างนี้ เห็นอย่างนี้ ภิกษุนั้นควรหรือที่จะ
กล่าวว่า ชีพก็อันนั้น สรีระก็อันนั้น หรือว่าชีพอย่างหนึ่ง สรีระอย่างหนึ่ง บรรพชิตทั้งสองนั้น
กล่าวว่า ท่านผู้มีอายุ ภิกษุใดรู้อย่างนี้ เห็นอย่างนี้ ภิกษุนั้นไม่ควรจะกล่าวว่า ชีพก็อันนั้น
สรีระก็อันนั้น หรือว่าชีพอย่างหนึ่ง สรีระอย่างหนึ่ง ดูกรผู้มีอายุ เรารู้อย่างนี้ เห็นอย่างนี้
เราจึงมิได้กล่าวว่า ชีพก็อันนั้น สรีระก็อันนั้น หรือว่าชีพอย่างหนึ่ง สรีระอย่างหนึ่ง.
			  เจโตปริยญาณ
	ภิกษุนั้น เมื่อจิตเป็นสมาธิ บริสุทธิ์ผ่องแผ้ว ไม่มีกิเลส ปราศจากอุปกิเลส อ่อน
ควรแก่การงาน ตั้งมั่น ไม่หวั่นไหว อย่างนี้ ย่อมโน้มน้อมจิตไปเพื่อเจโตปริยญาณ เธอย่อม
กำหนดรู้ใจของสัตว์อื่น ของบุคคลอื่น ด้วยใจ คือ จิตมีราคะก็รู้ว่าจิตมีราคะ หรือจิตปราศจาก
ราคะก็รู้ว่าจิตปราศจากราคะ จิตมีโทสะก็รู้ว่าจิตมีโทสะ หรือจิตปราศจากโทสะก็รู้ว่าจิตปราศจาก
โทสะ จิตมีโมหะก็รู้ว่าจิตมีโมหะ หรือจิตปราศจากโมหะก็รู้ว่าจิตปราศจากโมหะ จิตหดหู่ก็รู้ว่า
จิตหดหู่ หรือจิตฟุ้งซ่านก็รู้ว่าจิตฟุ้งซ่าน จิตเป็นมหรคตก็รู้ว่าจิตเป็นมหรคต หรือจิตไม่เป็น
มหรคตก็รู้ว่าจิตไม่เป็นมหรคต จิตมีจิตอื่นยิ่งกว่าก็รู้ว่าจิตมีจิตอื่นยิ่งกว่า หรือจิตไม่มีจิตอื่นยิ่งกว่า
ก็รู้ว่าจิตไม่มีจิตอื่นยิ่งกว่า จิตเป็นสมาธิก็รู้ว่าจิตเป็นสมาธิ หรือจิตไม่เป็นสมาธิก็รู้ว่าจิตไม่เป็น
สมาธิ จิตหลุดพ้นก็รู้ว่าจิตหลุดพ้น หรือจิตไม่หลุดพ้นก็รู้ว่าจิตไม่หลุดพ้น ดูกรผู้มีอายุ ภิกษุใด
รู้อย่างนี้ เห็นอย่างนี้ ภิกษุนั้นควรหรือที่จะกล่าวว่า ชีพก็อันนั้น สรีระก็อันนั้น หรือว่าชีพ
อย่างหนึ่ง สรีระอย่างหนึ่ง บรรพชิตทั้งสองนั้นกล่าวว่า ท่านผู้มีอายุ ภิกษุใดรู้อย่างนี้ เห็นอย่างนี้
ภิกษุนั้นไม่ควรจะกล่าวว่า ชีพก็อันนั้น สรีระก็อันนั้น หรือว่าชีพอย่างหนึ่ง สรีระอย่างหนึ่ง
ดูกรผู้มีอายุ เรารู้อย่างนี้ เห็นอย่างนี้ เราจึงมิได้กล่าวว่า ชีพก็อันนั้น สรีระก็อันนั้น หรือว่า
ชีพอย่างหนึ่ง สรีระอย่างหนึ่ง.
